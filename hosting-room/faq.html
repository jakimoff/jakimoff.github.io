1. General technical issues.

do You support the ability to assign multiple IP addresses to the dedicated one (dedicated) or virtual server (VDS)?
You can order up to 8 IP addresses on a dedicated (dedicated) or virtual server (VDS).

What control panel do you provide services?
- the management is through the hosting panel ISPmanager
- managing virtual servers (VDS) as the iron (restart, stop, etc.) is carried out through VDSmanager, to administer the VDS You can install the prepared images with built
-in and customized panels VestaCP (free) or ISPmanager 5 Lite (surcharge) upon request
- the management of dedicated servers as the hardware (restart, stop, etc.) is carried out through DCImanager, to administer the VDS You can install the prepared images with built
-in and customized panel ISPmanager 5 Lite (surcharge) upon request You can put any control panel, which is considered necessary as we provide root access on the server.

You provide the service of data backup (backup)?
No, but we are working on it. In the near future You will be able to get ftp repository for backups of all the information You need. For virtual servers (VDS) there is implicit redundancy through the mirroring of the repositories. And to save the machines state prior to important changes, You can use the snapshots (snapshot). We have not yet determined whether to enable the ability to create backups in the control panel server, because for a correct and complete copy of Your virtual machine, it is necessary to suspend that will interrupt the running of the services. We recommend You to use for free data backup virtual disks, such as Google Drive, One drive, Mail.ru disc, BOX, etc.

How can I change my IP address?
You can order additional IP on Your server. This is a paid service. Also, You can delete your server and may, when creating a new server You will get a different IP. The removal of the paid server does not return the money to Your account.

I get a dedicated "white" static IP address (dedicated static ip) for the ordered service?
the IP address given to You dedicated or virtual server (dedicated, VPS) is routable on the Internet and is assigned directly to the interface of Your machine without the use of any of the translations (NAT) static and reserved until You delete it. Backup server does not guarantee maintaining the same IP address that was assigned to You. Virtual hosting (shared hosting) by itself implies that many sites are under one IP, because in that case, Your IP will not be allocated to You personally.

What are your security settings (security settings) by default (ports, firewalls)?
By default, all ports open, no restrictions from our side. We encourage customers to set their own security settings and to ensure the security of their services.

do You offer hosting or dedicated (dedicated) and virtual server (VDS) on Windows?
Yes. We are a Microsoft SPLA and ready to provide You with licensed software of Microsoft, in particular Windows 2012 standard, Windows 2008 standard.

What is your architecture and technical implementation?
Despite the low prices, our architecture is similar failover clustering (high-availability cluster), which is built using tested for years in production of open and commercial software (software) and iron (hardware).
- All servers are running on CentOS 6, which ensures a homogeneous environment and simplifies administration and security.
- All services are run on clusters.
- All cluster standard.
- Standard cluster is a node-based server Dell c6100 server with 6-core Intel Xeon processors and storage based on Supermicro servers with 6-core Intel Xeon and SSD RAID 5 controller based on LSI.
- Cluster is served by switches, Juniper and IBM, and is connected via 10Gbps uplink.
- Each node is connected with two 1Gbps interfaces to provide fault tolerance for the physical link and provides the ability to create internal virtual links between virtual machines to 40Gbps.
- Storage connected by iSCSI 10Gbps interfaces.
