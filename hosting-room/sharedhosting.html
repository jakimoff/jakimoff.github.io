Shared Hosting

Type of hosting where many web sites are located on the same web server. This is the most economical type of hosting is suitable for small projects.

Shared Hosting

There are two basic methods to implement access to web sites:

  • named (shared IP hosting), when all websites use the same public IP address. According to the HTTP/1.1 web browser in the request to the web server specifies the domain name of the website in the Host field of the header of the current request, and the web server uses it for the correct execution of the query and copies this name into the cell [HTTP_HOST] superglobal array $_SERVER.
  • by IP address (dedicated IP hosting) at which each web site has its own IP address and the web server has multiple physical or virtual network interfaces.

In this type of hosting is limited

  • Disk space
  • The number of monthly traffic.
  • The number of websites you can host hosting as a service.
